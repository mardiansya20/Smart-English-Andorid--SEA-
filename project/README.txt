   Sea adalah sebuah aplikasi media pembelajaran yang berbasis mobile dengan android. SEA ini digunakan dalam bentuk offline dengan beberapa fitur- fitur sebagai pelengkapnya. Fitur- fitur tersebut yaitu: pemberian materi yang sangat membantu para pengguna dan  adanya evaluasi sebagai latihan untuk para pengguna agar lebih memahami dalam materi sebelumnya. Kehebatan dalam aplikasi ini adalah mampu memberikan suara kepada pengguna. Jadi aplikasi ini tidak hanya mengandalkan para pengguna untuk membaca tapi juga mengerti akan pelafalan atau pengucapan berbahasa inggris.  Dalam materi dibahas tentang penggunaan 16 tenses serta memberikan bacaan, contoh , dan pembahasan soal.  
Nama SEA (Smart English Andorid) diambil dari artinya yang berarti bahasa inggris pintar dengan meggunakan android sebagai  sistem ooerasi yang digunakan. Aplikasi ini memang bertujuan untuk menjadi media pembelajaran bahasa inggris. Aplikasi ini akan mengubah mindset masyarakat tentang cara belajar bahasa inggris yang awalna berbasis media cetak menjadi mobile phone dan akan mempermudah proses belajar menjadi pembelajaran mandiri berbasis mobilephone.
Untuk membuat aplikasi ini diperlukan sebuah aplikasi yang mendukung pembuatan media pembelajaran berbasis mobile yang digunakan untuk membuat aplikasi ini adalah memakai software desktop yang bernama Eclipse.  
Eclipse adalah sebuah IDE  (Integrated  Development Environment) untuk megembangkan perangkat lunak dan dapat dijalankan di semua platform.  Eclipse sangat digemari karena gratis dan open source. Eclipse juga dapat dikembangkang oleh para pengguna dengan membuat komponen yang disebuat plug-in.

2. Panduan Instalasi (singkat).
   Selain melalui Android Market, anda dapat menggunakan komputer anda untuk mengunduh game atau aplikasi lalu menginstallnya di handphone Android anda. Cara ini berguna untuk menginstall game atau aplikasi yang tidak tersedia di Android Market.
Berikut cara install aplikasi Android melalui computer:
Pertama: unduh dulu aplikasi media pembelajarannya di url diatas.
Kedua: Hubungkan ponsel ke komputer menggunakan kabel data lalu tranfer file APK dari komputer ke SD card. Bila tidak memiliki kabel data, gunakan card reader untuk mentransfer file APK dari komputer ke SD card. Anda dapat juga menggunakan Bluetooth untuk melakukan transfer file.
Ketiga: buka aplikasi file manager (My Files / “file Saya” / File Browser) di handphone Android > cari file APK yang tadi di transfer > klik pada file APK untuk melakukan instalasi. Apabila pada handphone Android anda tidak terdapat aplikasi file manager, anda dapat mengunduhnya terlebih dahulu dari Android Market. Salah satu aplikasi file manager yang terkenal adalah Astro File Manager. Untuk mencari aplikasi file manager menggunakan Android Market, anda hanya cukup mengetik kata kunci “file manager” pada kolom pencarian.
Apabila aplikasi tidak dapat install, cek pengaturanya:
1. Pergi ke Settings (Pengaturan)
2. Klik Applications (Aplikasi)
3. Beri tanda centang “Unknown Source” (Sumber tidak dikenal)
